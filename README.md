# 🏏 IPL Analytics Dashboard

An interactive Tableau dashboard analyzing Indian Premier League (IPL) matches from 2008–2024. The project provides insights into team performance, venue statistics, toss decisions, match outcomes, and player achievements using cleaned cricket datasets.

---

## 📌 Project Overview

This project transforms raw IPL match and ball-by-ball delivery data into an interactive business intelligence dashboard using Tableau.

The dashboard helps users explore:

- Overall IPL statistics
- Season-wise match trends
- Team performance
- Venue analysis
- Toss decision insights
- Win type distribution
- Orange Cap & Purple Cap winners
- Top run scorers and wicket takers

## 📈 Dashboard Insights

### Overall Tournament Statistics

- 1,095 IPL Matches
- 17 Seasons
- 19 Teams
- 58 Venues
- 347,756 Total Runs
- 12,950 Total Wickets

### Business Insights

- Match count has steadily increased across IPL seasons.
- Toss winners choose to field more frequently.
- Certain venues host significantly more matches than others.
- Winning the toss does not guarantee winning the match.
- Matches are more frequently won by wickets than by runs.
